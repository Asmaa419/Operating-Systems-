Types of Operating system:

1.Batch Operating System:-Some computer processes are very lengthy and time-consuming. To speed the same process, a job with a similar type of needs are batched together and run as a group.

2.Multi-Tasking/Time-sharing Operating systems:-Time-sharing operating system enables people located at a different terminal(shell) to use a single computer system at the same time. The processor time (CPU) which is shared among multiple users is termed as time sharing.

3.Real time OSA:- real time operating system time interval to process and respond to inputs is very small. Examples: Military Software Systems, Space Software Systems..

4.Distributed Operating System:-Distributed systems use many processors located in different machines to provide very fast computation to its

5.Network Operating System:- Network Operating System runs on a server. It provides the capability to serve to manage data, user, groups, security, application, and other networking functions.
